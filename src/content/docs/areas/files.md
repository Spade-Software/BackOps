The **Files** tab allows you to upload and manage documents associated with a specific Area‚Äîeverything from Tech Packs to site maps and advancing notes.

---

### üß∞ What You Can Do

- **Upload Files**
  Click the **Upload File** button in the top right to add PDFs, images, spreadsheets, and more.

- **View File Metadata**
  Each file entry shows:

  - File type (icon)
  - File name
  - Upload date
  - Associated collaborator (if applicable)
  - Uploading user

- **Manage Files**
  Use the three-dot menu on each row to:

  - Rename files
  - Download files
  - Delete files

- **Search & Filter**
  Use the search bar to quickly find files in areas with heavy documentation.
